## Features 

- **Add Tasks**: Quickly add tasks to your ToDo list using the input field and "Add" button.
- **Edit Tasks**: Click on edit icon of a task to edit its content.
- **Delete Tasks**: Remove tasks using "Delete" icon you no longer need.
- **Persistent Storage**: Your tasks and progress are saved in **localStorage**, so even after refreshing the browser, your ToDo list remains intact.
- **Progress-Based Rewards**: Unlock cat memes as you complete tasks.


---

## How It Works 

1. Enter a task in the input field and click **Add**.
2. Tasks will appear in the list below.
3. Complete tasks by marking them done.
4. As you progress, cat memes will appear according to the completion percentage.
5. Double-click a task to **edit** it.
6. Click the delete button to **remove** a task.
7. All your tasks and cat memes are saved in your browser's **localStorage**, so your progress is retained across sessions.
